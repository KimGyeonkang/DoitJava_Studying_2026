# DoitJava_Studying_2026

## Overview(프로젝트 개요)
1. 기간: 2025.12.30 ~
2. 참여 목적
  * 일본 취업 교육에서 가장 많이 다뤄지는 기본 언어인 JAVA의 기초를 전체적으로 이해하며 실습
  * 우선 프로그래밍 자체와 자바라는 언어에 익숙해지기 위함.
  * 더불어, 독학으로는 쉽지 않은 꾸준한 공부 습관 형성에도 도움을 받고자 함.
4. 목표
  * 약 6주 안에 스터디 도서(아래 Reference 참고)를 완독하며 언어의 기초를 익히며
  * 책 한 권을 성공적으로 읽어냈다는 성취감을 맛보고
  * 앞으로도 이어질 프로그래밍, CS 공부와 일본 IT 취업 준비에 동기를 부여하고자 함.
