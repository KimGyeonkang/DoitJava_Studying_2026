# DoitJava_Studying_2026

## Overview(프로젝트 개요)
0. 스터디 명칭: Do it! 기초 언어 스터디 - 자바 팀(이지스퍼블리싱 출판사 개최)
1. 기간: 2025.12.30 ~ 2026.02.13
2. 참여 목적
  * 일본 취업 교육에서 가장 많이 다뤄지는 기본 언어인 JAVA의 기초를 전체적으로 이해하며 실습
  * 우선 프로그래밍 자체와, 자바라는 프로그래밍 언어에 익숙해지기 위함.
  * 더불어, 독학으로는 쉽지 않은 꾸준한 공부 습관 형성에도 도움을 받고자 함.
3. 목표
  * 약 6주 안에 스터디 도서(아래 Reference 참고)를 완독해 언어의 기초를 익히며(완독은 성공, 기초는 어느 정도 눈에 익었다 정도까지는 공부, 구체적인 활용 면에서는 꾸준히 개선 요함)
  * 책 한 권을 성공적으로 읽어냈다는 성취감을 맛보고(달성)
  * 앞으로도 이어질 프로그래밍, CS 공부와 일본 IT 취업 준비에 동기를 부여하고자 함.(어려움도 많았지만 이쪽도 성공한 듯)
4. 결과와 배운 점
  * 6주 동안 자바 기본서 한 권을 완독. 혼자서는 쉽지 않았겠지만 스터디를 통해 완수할 수 있었음.
  * 아직 모든 기초를 이해했다고는 할 수 없지만, 최소한 '이거 스터디하면서 들어봤다'와 간단한 개념(변수가 뭔지, 조건문과 반복문의 기본 구조가 뭔지 이 정도)은 이해하고 설명할 수 있을 정도로 공부했다고는 할 수 있음.
  * 자바의 개념이 꽤나 방대함을 깨닫고, 추후 직접 실전을 겪어보며 그때그때 필요한 개념 위주로 복습해야겠다는 방향성을 떠올림. -> 모든 개념을 다 섭렵하기에는 양도 너무 많고 공부에 대한 흥미도 저하됨을 느낌.
5. 이번 스터디에 참여하며 느낀 점이나 공부 내용 등은 블로그에 자세히 정리해 둠.
